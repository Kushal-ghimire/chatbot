# Chatbot Project

This project consists of two parts: Backend and Frontend.

## Backend

### Installation

1. Navigate to the `backend` directory:

    ```bash
    cd backend
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

### Running

- Start the backend server:

    ```bash
    npm start
    ```

    For development with nodemon:

    ```bash
    npm run dev
    ```
